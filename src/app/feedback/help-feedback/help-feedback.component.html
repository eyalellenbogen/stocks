Welcome to TechStocks.
This app allows you to view stock price information for 5 big tech companies. Feel free to click some buttons.

Zooming

You can zoom in by selecting an area on the graph.

Errors

To see the error modal make more than 5 requests to the server within one minute. If you can't reach the limit, try refreshing the page because the app caches requests. Once you do that, the API key I used will reach its rate limit and an error message will appear.
